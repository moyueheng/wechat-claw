# 明天油价大幅上调！加满一箱油预计多花近40元

**来源**: [东方财富](https://finance.eastmoney.com/a/202603093665853677.html)  
**发布时间**: 2026-03-09 10:41  
**新闻ID**: 202603093665853677

【明天油价大幅上调！加满一箱油预计多花近40元】3月9日24时，国内成品油调价窗口将再度开启。记者从隆众资讯最新获悉，预计每吨汽油上调695元，每吨柴油上调670元，折合每升89号上调0.50元、92号上调0.53元、95号上调0.56元、0号上调0.57元。该预测涨幅较上周五进一步上调，以70升的油箱计算，预计私家车主加满一箱油贵近40元（上周五预计多花27元左右）。
