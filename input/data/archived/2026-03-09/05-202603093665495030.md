# 一年来全国人大常委会组织对影响民营经济发展等规范性文件开展集中清理

**来源**: [东方财富](https://finance.eastmoney.com/a/202603093665495030.html)  
**发布时间**: 2026-03-09 09:09  
**新闻ID**: 202603093665495030

【一年来全国人大常委会组织对影响民营经济发展等规范性文件开展集中清理】全国人大常委会工作报告显示，一年来，全国人大常委会依法履行备案审查职责，做到有件必备、有备必审、有错必纠。对报送备案的2218件行政法规、监察法规、地方性法规、自治条例和单行条例、司法解释、特别行政区本地法律进行审查，对公民、组织提出的6705件审查建议逐一研究并依法反馈，组织对影响民营经济发展、不平等对待企业等规范性文件开展集中清理。
