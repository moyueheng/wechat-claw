# 油价四年来首度破百！美能源部长安抚市场：最多不会超过几周

**来源**: [东方财富](https://finance.eastmoney.com/a/202603093665495510.html)  
**发布时间**: 2026-03-09 09:10  
**新闻ID**: 202603093665495510

【油价四年来首度破百！美能源部长安抚市场：最多不会超过几周】在伊朗战争持续推高国际油价之际，美东时间周日，美国能源部长克里斯·赖特（ChrisWright）表示，近期油价的大幅上涨反映了与伊朗战争相关的暂时“恐慌溢价”，这种现象不太可能持续下去，因为全球能源供应依然充足。
